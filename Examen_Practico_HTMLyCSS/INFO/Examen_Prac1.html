HTML

Lenguaje de programación que permite insertar y ordenar información presentada en una página web estática.





CSS

  1. Externo
    • Menor prioridad
    • Se incluye con el uso de la etiqueta "link", haciendo referencia a un archivo .css externo, en el "head"
  2. Interno
    • Mayor prioridad que externo, menor que en línea
    • Se incluye con el uso de la etiqueta "style" dentro de "head"
  3. En línea
    • Mayor prioridad
    • Se incluye con el atributo "style=" en la etiqueta que se desea emplear
